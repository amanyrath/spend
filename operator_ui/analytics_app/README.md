# Analytics App - Operator Dashboard

This is the React-based analytics dashboard for SpendSense operators. It provides detailed visualizations and metrics for monitoring recommendation quality, persona distributions, and system performance.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Features

- Real-time analytics overview
- Persona distribution charts
- Weekly persona trends
- Success metrics by persona
- Safety indicators monitoring
- Active user tracking

## Technology Stack

- React 18
- TypeScript
- Recharts (data visualization)
- Tailwind CSS (styling)
- Vite (build tool)
